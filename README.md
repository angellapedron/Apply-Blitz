# ApplyBlitz - Job Application Management Platform

A modern web application to help users create tailored CVs and Cover Letters for job applications with AI-powered content generation, multilingual support (English/German), and application tracking.

## 🚀 Tech Stack

### Frontend

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

### Backend

- **Framework**: FastAPI (Python)
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy (async)
- **Authentication**: FastAPI-users
- **Payments**: Stripe
- **AI**: OpenAI API
- **Error Tracking**: Sentry
- **Deployment**: Render
